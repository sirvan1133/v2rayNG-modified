# v2rayNG Modified

نسخه شخصی‌سازی شده v2rayNG با قابلیت‌های اضافه

## تغییرات اعمال شده

### 1. نوار مصرف ترافیک
- نمایش نوار مصرف ترافیک در بالای صفحه (مخصوص هر گروه/ساب)
- نمایش درصد مصرف، حجم مصرف شده، حجم باقی مانده، کل ترافیک و روزهای باقی مانده
- رنگ نارنجی هماهنگ با تم v2ray

### 2. بررسی خودکار آپدیت
- بررسی خودکار نسخه جدید از گیت‌هاب هر ۲۴ ساعت
- نمایش دیالوگ در صورت وجود نسخه جدید

### 3. رفع باگ‌ها
- رفع مشکل UnsatisfiedLinkError برای libhev-socks5-tunnel
- رفع مشکل حساسیت به حروف بزرگ و کوچک در هدر Subscription-Userinfo
- رفع مشکل محاسبه تاریخ انقضا (تبدیل ثانیه به میلی‌ثانیه)
